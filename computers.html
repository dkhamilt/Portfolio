Hardware Basics

Hardware is all the physical components of a computer, such as the motherboard (MOBO) or hard drive(HD).

Motherboard

The Motherboard is the main circuit board found in a computer. It is where most of the highly important components connect to, such as the memory, the cpu, and also things like keyboards and mice. Without the motherboar dyou would just have a box full of parts.

CPU

The CPU, or central processing unit is the piece of hardware responsible for performing tasks specified by computer programs. Some of these tasks include basic arithmetic as well as logical functions.

Hard Drive

The Hard Drive, (AKA Hard Disc Drive, HD, or HDD) is the main storage device on a computer. They store data ranging from Operating Systems (AKA OS) to images and games.

Component Communication

Component Communication is the process that different parts of a computer use to transmit informaton between themselves. Components communicate through another component called a bus. A bus transmits data just as a real life bus transmits people between places.

Peripherals

Peripherals can be sorted into two catergories, input and output. Input devices such as a keyboard can be used to give commands to the computer, and output devices such as monitors and speakers are used to display the results of these commands. As mentioned some peripherals include keyboards, which are basically large arrays of buttons (generally with letters on them) used to input data, mice, which are devices that allow you to navigate and select things freely on a two dimensional plane, and monitors, which are visual displays that can display things such as text and images.
