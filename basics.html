What is HTML

HTML, or Hyper Text Markup Language is a programming language
that is often used to construct websites. This website was constructed
using HTML, as well as CSS (Cascading Style Sheets, another
programming language used for websites. You will find out more
about CSS further down on the page.) Programming in HTML
requires the use of elements, which require tags, which are all
constructed of matching greater than and less than signs. You will
find out more about tags and elements later.

What is CSS

CSS, or Cascading Style Sheets is a programming language used to
style websites. While HTML controls the content on a website, CSS
is what makes it presentable to the user.

Without CSS
and With CSS


CSS can be used to change colors, add animations, adjust fonts, and
everything inbetween.

What are Tags and Elements

Elements are what HTML code consists of. Tags consist of angle brackets
with keywords in between them, and then content in between tags. Example
element syntax is

<keyword>content</keyword> If you want to add a paragraph to the page, you need to create an element that
tells the computer to read it as a paragraph. The way you do this is using the

tag. The full syntax would be

<p>(What you want to write)</p> At the bottom of the page is a table showing many example tags.

CSS Syntax and Attributes

In CSS, to control how the content from the HTML is presented,
you need to use the following Syntax.

(Object you want to change){
(How you want it to change)
}

For example, if you wanted to make all your paragraphs have blue
text and a green background, you would do

p{
color:blue; (Pay attention to semicolons and colons)
background-color:green;
}

If you only want to change one paragraph, or a specific selection
of paragraphs, that’s where attributes come in. Attributes contain
IDs and Classes. ID’s specifically describe one element of HTML.
For example, you might assign your first paragraph an ID of
“intro”, to show that it is the introduction paragraph. Classes
describe many elements. For example, you might give all of your
paragraphs in a certain section of your website a class of “section1”.

Now that you know what attributes are, it’s time to learn how to use them.
A paragraph with an ID of intro would be formatted like this:

<p id=”intro”>content</p>,and a paragraph with a class of section1 would be formatted like this: <p class=”section1”>content</p>. Now you need to know how to use CSS to change the appearance of
elements with specific IDs and classes. The standard CSS syntax remains
mostly the same, but you lead the names of IDs with #s, and the names
of classes with a period. So the formatting looks like this:
#ID{
(Changes Made)
}
And:
.class{
(Changes Made)
}
